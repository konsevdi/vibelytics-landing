# Vibelytics SR007 Summary

Vibelytics is the AI launch copilot for live culture.

It helps promoters, venues, creators, and brands decide which events to launch before capital is committed: where to launch, at what capacity, at what price, with which sponsor, and with what launch story.

## Core workflow

1. Enter artist, city, venue capacity, target date, and sponsor category.
2. Review demand score, venue fit, pricing band, sponsor fit, and risk.
3. Receive a launch recommendation: go, adjust, or do not launch.
4. Use generated launch copy to shape the first announcement.

## Submission scope

This web submission is static and uses synthetic demo data only. It includes a polished landing page and a clickable pilot page. No backend, API keys, or external services are required.

SceneQuest is a future activation module after the launch decision layer.
